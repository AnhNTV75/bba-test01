# Playwright là gì?
## Khái niệm
- Là 1 framework
- Tiền thân là Puppeteer
## Ưu điểm
- Cross browser -> hỗ trợ các trình duyệt phổ biến: Chrome, Edge, Firefox, Safari
- Cross platform -> Code 1 lần, chạy trên các trình duyệt phổ biến: Windows, Linux, MacOS
- Auto waiting, auto-retry assertion giúp giảm flaky tests (lúc pass, lúc fail)
- Report đầy đủ thông tin Pass/ Fail
- Code gen

# Đưa code lên GitHub
- Khởi tạo repo local -> git init
- Tạo repo GitHub & liên kết tới repo local -> git remote add origin <url>
- Thêm git vào staging -> git add .
- Commit file -> git commit -m <message>
- Push code -> git push origin main
